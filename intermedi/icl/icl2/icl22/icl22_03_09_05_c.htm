Imprimeix

Observa els complements directes d'aquestes oracions i com s'han substituït per pronoms:

Si el CD és definit, els pronoms que el substitueixen són: el, la, els, les
Si el CD és indefinit, el pronom que el substitueix és: en
Si el CD és neutre o una oració, el pronom que el substitueix és: ho


Activitat 25

Identifica els complements directes. Controla si són definits, indefinits o neutres. Després, repeteix l'oració substituint el CD pel pronom adequat.
