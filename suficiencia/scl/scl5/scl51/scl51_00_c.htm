Imprimeix

L'ortografia estudia la relació que s'estableix entre els sons d'una llengua i les lletres que fem servir per representar aquests sons en l'escriptura.

Tal com veurem, no sempre hi ha correspondència exacta entre so i grafia. Algunes grafies poden representar dos o més sons. Alguns sons poden ser representats per més d'una grafia diferent. I algunes grafies poden no tenir so.

Exemples:

La lletra a pot representar el so de vocal oberta [a] (pa) o bé el so de
vocal neutra [] (camí).

 

El so [z] es representa amb les lletres s (casa) o bé z (zeta).


El català, com totes les llengües, té dos tipus de sons: les vocals, que tenen un paper més destacat, i les consonants. Els sons consonàntics no poden formar síl·labes sols, sense el suport d'una vocal.

Tots els sons, ja siguin vocàlics o consonàntics, són representats per grafies o lletres que, en català, tenen unes correspondències similars a la resta de llengües romàniques.

El conjunt de lletres o grafies s'anomena alfabet o abecedari.
