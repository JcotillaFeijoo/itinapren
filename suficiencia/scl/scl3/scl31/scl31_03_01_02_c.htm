solució

Ordres singular

Ordres plural

1. Nena, menja’t la sopa!

Nenes, mengeu-vos la sopa!

2. Nen, neteja’t les dents!

Nens, netegeu-vos les dents!

3. Nena, renta’t les mans!

Nenes, renteu-vos les mans!

4. Nen, respecta els grans!

Nens, respecteu els grans!

5. Nen, calla!

Nens, calleu!
